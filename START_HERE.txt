🎯 HOW IT WORKS

  1. Monitors Gmail inbox for CSV attachments (every 2 hours)
  2. Parses CSV with user data and image generation prompts
  3. Generates personalized emails using your LLM API
  4. Sends emails via Gmail SMTP
  5. Tracks everything in SQLite (no duplicates)


✨ KEY FEATURES

  ✅ Your custom LLM API integration (pre-configured)
  ✅ Gmail IMAP reader for CSV attachments
  ✅ Personalized email generation
  ✅ Email SMTP sender with anti-spam headers
  ✅ SQLite deduplication
  ✅ Cron scheduling (configurable)
  ✅ Rate limiting between emails
  ✅ Comprehensive logging
  ✅ Graceful error handling


📂 PROJECT STRUCTURE

  src/
  ├── index.ts ................. Main entry + cron
  ├── config/env.ts ............ Config validation
